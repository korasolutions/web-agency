# Nexora Solutions - Web Corporativa 3D

##Una web corporativa moderna con efectos 3D avanzados, diseñada para una agencia digital especializada en transformación digital, desarrollo web y automatizaciones con IA.

## Características Principales
 ### Diseño Visual
Tema Oscuro Minimalista: Paleta de colores azules apagados (#3a5a78) sobre fondo oscuro (#0a0a0a)

Tipografía Moderna: Fuente Inter para máxima legibilidad

Espacio Generoso: Diseño con mucho espacio en blanco (espacio negativo)

Responsive Total: Adaptado a desktop, tablet y móvil

## Efectos 3D y Animaciones
### Sistema de Partículas: 50 partículas flotantes con animaciones personalizadas

### Mouse Tracking: Elementos que responden al movimiento del cursor

### Parallax Avanzado: Múltiples capas con efectos de profundidad

### Transformaciones 3D: Rotaciones y perspectivas en hover

### Animaciones de Entrada: Efectos fade-in, slide y rotate al hacer scroll

### Hover Effects: Interacciones sutiles pero impactantes

## Tecnologías Utilizadas
### HTML5: Estructura semántica y moderna

### CSS3: Variables CSS, Grid, Flexbox, Animaciones, Transformaciones 3D

### JavaScript Vanilla: Sin frameworks pesados, código optimizado

### Font Awesome: Iconografía consistente

### Google Fonts: Fuente Inter para tipografía
